## 附录A：20条生存模式下的核心行动清单

（当你处于资源匮乏、前途未卜的“生存阶段”时，请对照此清单，检查你的每一个行动。）

### 心态与认知

1. 拥抱野蛮：承认自己一无所有，抛弃所有不必要的“体面”和“自尊”，唯一的目标是活下去。
2. 先开枪，后瞄准：停止过度规划，用最小成本的行动去测试市场，在反馈中迭代。
3. 聚焦现金流：忘掉估值和梦想，像守护生命一样守护你的现金流。任何不能在短期内带来正向现金流的业务，都要极度警惕。
4. 结果导向：不要在意过程有多辛苦、多“正确”，只用最终的、可量化的结果来衡量一切。

### 机会与客户

1. 先找买家，再找货：在投入任何资源开发产品前，先找到至少一个愿意为你的“解决方案”预付定金的客户。
2. 解决“痛点”，而非“痒点”：你的产品必须像“止痛药”，而不是“维生素”。它解决的是客户“没有不行”的紧急问题。
3. 做小池塘里的大鱼：选择一个极度细分的、被巨头忽视的“针尖市场”，集中所有火力，成为这个小领域的绝对第一。
4. 成为客户的“自己人”：不要把自己当成“卖家”，而是客户的“问题解决顾问”。深度参与他们的业务，赢得他们的信任。

### 产品与执行

1. MVP（最小可行产品）：用最丑陋、最简陋但能验证核心功能的产品去接触市场，速度永远比完美重要。
2. 抄袭、模仿、再超越：不要羞于学习和模仿市场上已经验证成功的模式，在模仿的基础上，做出你的“微创新”。
3. 单点极致：不要追求功能的“大而全”，把一个用户最核心的痛点，做到比所有竞争对手都好10倍。
4. 手动优于自动：在早期，用大量的人工操作来替代昂贵的自动化系统，保持模式的灵活性和低成本。

### 团队与人脉

1. 寻找“野狼”，而非“绵羊”：你的早期伙伴必须是和你一样饥饿、自驱、渴望胜利的“合伙人”，而不是朝九晚五的“打工者”。
2. 价值先行：建立人脉的唯一方法，是持续地为别人提供价值，让自己成为一个“值得被认识”的人。
3. 找到比你聪明的人，然后闭嘴：在专业领域，充分授权，并为他们的错误买单。
4. 你的愿景是团队唯一的弹药：在物质匮乏时，你必须成为团队信心的唯一来源，反复“布道”，庆祝每一个微小的胜利。

### 个人管理

1. 出售你的时间两次以上：把你的经验和技能，产品化、系统化，打破“一份时间卖一份”的线性收入模式。
2. 保持极度专注：在同一时间，只做最重要的一件事。学会对99%的“机会”说不。
3. 保护你的精力：精力比时间更宝贵。确保充足的睡眠、健康的饮食和定期的运动，这是你持续战斗的基础。
4. 记录与复盘：每天记录你的关键行动和结果，每周进行一次复盘。从自己的错误中学习，是成长最快的方式。

## 附录B：10个识别机会与陷阱的关键问题

（当你面临一个新的商业机会、一个合作邀约或一个重大决策时，请用这10个问题来审视它。）

1. 时机问题：市场真的准备好了吗？我做这件事，是顺风，还是逆风？是“先驱”，还是会成为“先烈”？
2. 痛点问题：这件事解决的是一个“真实且迫切”的痛点，还是一个“想象中且不重要”的痒点？是否有人已经愿意为此付费？
3. 牌局问题：这个赛道上，最强的玩家是谁？我的核心优势是什么？我是来打一场“优势仗”，还是一场“自杀式”的攻击？
4. 不对称性问题：这件事，是“亏损有限，盈利无限”的机会，还是“盈利有限，亏损无限”的陷阱？最坏的结果，我能否承受？
5. 复利问题：这件事，能否积累下可复用的“资产”（如品牌、数据、核心能力）？它的价值，是会随着时间而“增值”，还是会“消耗”掉？
6. 现金流问题：它需要我投入多少前期成本？多久能看到正向的现金流回报？它会改善我的财务状况，还是会让我陷入更大的风险？
7. 专注度问题：做这件事，是会增强我的“主营业务”，还是会分散我宝贵的精力，让我“多线作战”？它符合我的长期战略吗？
8. 可复制性问题：这次成功，能否被“系统化”、“产品化”，从而实现规模化扩张？还是一个无法复制的、一次性的“偶然”？
9. 价值链问题：在这个机会中，我处于价值链的哪个位置？是核心的、不可替代的“价值创造者”，还是随时可能被绕开的“中间商”？
10. 退出问题：如果事情进展不顺，我的“退出机制”是什么？我是否为这个“机会”的失败，准备了B计划和Z计划？

## 附录C：个人成长每日自省清单

（建议每天睡前，花10-15分钟，对照此清单，进行一次诚实的自我对话。这不是为了批判，而是为了觉察和迭代。）

### 关于行动与结果

1. 今日最重要的成果是什么？我是否完成了那件“吃掉青蛙”的最重要的事？
2. 今日最大的失败/错误是什么？在这个错误中，我的责任是什么？我能提炼出什么教训，以避免明天再犯？
3. 今日的时间/精力，主要花在了哪里？这些投入，是否与我的长期目标一致？是否存在大量的“伪工作”和“内耗”？

### 关于认知与学习

1. 今日我学到了什么新知识/新观点？它对我的既有认知，带来了什么冲击或补充？
2. 今日我是否主动寻求了“负反馈”？对于听到的批评，我是否做到了“不辩解，只记录”？
3. 今日我的哪个决策，是基于“情绪”，而非“理性”？当时我的内心状态是怎样的？未来如何避免？

### 关于心态与秩序

1. 今日我是否被“压力”所控制？我是如何应对的？我是否成功地将它转化为“燃料”？
2. 今日我是否因为“过去”而烦恼，或因为“未来”而焦虑？我有多少时间，是真正地“活在当下”？
3. 在无人监督时，我的言行和念头，是否守住了我的“核心价值观”？（慎独）
4. 感恩与展望：今日最值得我感恩的一件事（或一个人）是什么？基于今天的复盘，我明天需要做出的一项“微小但重要”的改进是什么？

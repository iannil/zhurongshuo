序言：你不是在创业，你是在打仗

2025.11.16

请先忘掉你读过的所有关于创业的书。

忘掉那些在窗明几净的咖啡馆里，一边品尝着手冲咖啡，一边用最新款笔记本电脑敲下商业计划书的优雅画面。忘掉那些在TED演讲台上，用十五分钟描绘一个改变世界、估值百亿的宏大愿景。忘掉那些西装革履、在顶级风投机构的会议室里，用精美的PPT侃侃而谈的商业精英。

忘掉它们。至少现在，请把它们从你的脑海里彻底清除。

因为那些，是属于胜利者的故事，是战争结束后的授勋仪式。而你，我的朋友，战争才刚刚开始。你甚至还不是一名士兵，你只是一个被空投到未知战场、口袋里只有一把小刀和半块干粮的幸存者。

你不是在创业。

你是在打仗。

一场为了生存、为了尊严、为了夺回人生主动权的，残酷而孤独的战争。这场战争没有后援，没有规则，甚至没有明确的敌人，因为你周围的一切——时间、账单、同行的漠视、家人的不解、甚至是你内心的怀疑——都是你的敌人。

这本书，就是写给像你一样，身处战壕，满身泥泞，却依然眼含杀气的人。它不是一本商业教科书，它是一本从真实战场上用鲜血和教训写成的野战手册。

它的核心思想，就是书名里的四个字：野蛮秩序。

一、为什么是“野蛮秩序”？

“野蛮”这个词，听起来不那么悦耳。它通常与粗鲁、无序、缺乏文明联系在一起。但在我们即将踏入的这场战争中，“野蛮”是一种赞美。它代表着一种最原始、最强大、最不受束缚的生命力。

想象一下，一只在五星级酒店里被精心喂养的波斯猫，和一只在城市最混乱的后巷里长大的野猫。波斯猫优雅、美丽，习惯了定时定点的美食和柔软的猫砂盆。它的世界是确定的，安逸的。而那只野猫，它可能毛色杂乱，身上带着伤疤，眼神里永远保持着警惕。但它知道哪里的垃圾桶能翻出食物，知道如何躲避车辆和恶犬，知道如何在一群同类的竞争中抢到一口吃的，知道如何在严寒的冬夜找到一个避风的角落活下来。

在创业这场战争的初期，你不能做那只波斯猫。你必须成为那只野猫。

“野蛮秩序”，就是一种彻底抛弃幻想，直面现实的生存姿态。

它意味着，你必须抛弃“体面”的幻想。

在生存阶段，体面是最奢侈、也是最无用的东西。你可能需要去做那些你曾经看不起的“小生意”，去赚那些你觉得“没意思”的辛苦钱。你可能需要放下所谓的“专家”身段，亲自去发传单，去跟人讨价还价，去处理最琐碎的客户投诉。当你口袋里的钱只够支付下个月的房租时，思考“我的事业是否足够伟大”是一个会杀死你的问题。唯一重要的问题是：“我今天如何赚到钱，好让我明天还能继续战斗？”先别想着打造品牌，先想着填饱肚子。品牌是打赢战争后的战利品，不是你冲锋时需要扛着的旗帜。

它意味着，你必须抛弃“完美”的幻想。

完美的商业计划书，在接触到第一个真实客户后就会被撕得粉碎。完美的产品，在你资金耗尽时也无法推向市场。在街头，速度和迭代比完美重要一百倍。你的第一个产品可能很粗糙，你的第一次服务可能不尽人意，但这都没关系。重要的是你把它拿出去了，你得到了市场的真实反馈——无论是冷漠的拒绝还是真金白银的订单。街头的法则是“完成”远胜于“完美”。先开枪，再瞄准。先让车子跑起来，再在路上边跑边修。等待万事俱备，你等来的只会是万念俱灰。

它意味着，你必须抛弃“资源”的幻想。

不要再抱怨你没有资金，没有人脉，没有背景。那只后巷里的野猫，它有什么？它一无所有，但它也拥有一切——它拥有整个城市作为它的狩猎场。野蛮秩序的核心，就是一种“无中生有”的能力。你身边的一切，都是你的资源。别人眼里的垃圾，可能是你眼中的宝藏；大家扔掉的东西，可能蕴含着巨大的价值。你的资源不是你银行账户里的数字，而是你的眼睛，你的双腿，你的大脑，和你那颗拒绝认输的心。你要学会像一个拾荒者一样，在废墟中寻找可用的零件；你要学会像一个游击队员一样，利用现有的一切地形和工具，去打击装备精良的敌人。

“野蛮秩序”不是让你去做一个违法乱纪的恶棍，而是让你成为一个清醒的现实主义者。它是一种思维模式的彻底转变：从“我需要什么才能开始”，转变为“我拥有什么，以及我能用它们做什么”。它要求你像一个侦探一样，敏锐地观察人性、需求和被忽视的角落；它要求你像一个机会主义者一样，随时准备扑向任何一个可能让你活下去的机会，无论它看起来多么微小。

这就是“野蛮秩序”的真义。它不优雅，但极其有效。它是你在这场战争中，必须学会的第一课。

二、先有资格上牌桌，再想如何赢下牌局

战争不是一成不变的。从混乱的巷战到集团军的正面决战，其战术、战略和指导思想截然不同。同样，你的创业之战，也清晰地分为两个截然不同的阶段：生存阶段与胜利阶段。

混淆这两个阶段的打法，是99%的失败者共同犯下的致命错误。他们在需要不择手段活下来的时候，却在思考如何赢得漂亮；在应该排兵布阵、决胜千里的时候，却还在用游击战的思维打一枪换一个地方。

这本书的结构，就是围绕着这个“两阶段论”展开的。第一部分“生存法则篇”，是教你如何不惜一切代价活下来；第二部分“赢家心法篇”，是教你活下来之后，如何赢得整个牌局。

阶段一：生存阶段：不惜一切代价，留在牌桌上

想象一场德州扑克。你用身上最后的钱，买了一张门票，坐上了牌桌。桌上的玩家个个筹码堆积如山，他们谈笑风生，可以轻松地跟注、诈唬，甚至输掉几手也无伤大雅。而你，你的全部身家就是面前那几片薄薄的筹码。

在这个阶段，你的目标不是“赢”，而是“不下桌”。

你的每一次下注都必须极度谨慎。你不能玩那些高风险高回报的牌，你只能玩那些胜率最高的牌。你不能意气用事，不能被对手的挑衅激怒。你的唯一任务，就是通过无数次微小的胜利，一点一点地积累你的筹码，让你从牌桌上最危险的边缘位置，慢慢向中间移动。

这就是生存阶段的本质。在这个阶段：

现金流就是你的氧气。你做的一切决策，都必须围绕“现金”展开。一个项目，无论前景多么宏伟，如果它在短期内不能带来正向现金流，那它就是毒药。你要像沙漠中的旅人珍惜水源一样，珍惜你的每一分钱。

“先找到买家”是不可动摇的铁律。不要去创造一个你自认为伟大的产品，然后祈祷有人来买。这是波斯猫的思维。野猫的思维是：先找到一群饥饿的人，看清楚他们想吃什么，然后再去给他们找食物。哪怕你找到的只是残羹冷炙，但因为他们足够饥饿，他们会付钱给你。先有订单，再有生产；先有客户，再有服务。这是生存阶段的唯一捷径。

你的时间和精力是最大的成本。你要毫不留情地砍掉一切“无用功”。什么是无用功？任何不能在短期内帮你赚到钱、或者帮你找到能赚钱的客户的事情，都是无用功。参加那些务虚的行业会议、写一份无人问津的商业计划书、为了一个无关紧要的细节反复修改产品……这些都是在消耗你宝贵的弹药。

不要瞧不起任何一笔小钱。在你快要饿死的时候，一个馒头就是全世界。生存阶段，积少成多是唯一的逻辑。每一笔小交易，都在为你增加筹码，都在为你争取多留在牌桌上一分钟的时间。正是这些你曾经看不上的“小钱”，最终构筑了你活下去的防线。

这个阶段是痛苦的、屈辱的、毫无美感可言的。你会在无数个深夜怀疑自己，会因为一笔小订单而欣喜若狂，也会因为一个客户的刁难而怒火中烧。但你必须熬过去。因为只有熬过这个阶段，你才真正获得了“上牌桌”的资格。你的筹码变多了，你不再是那个随时会出局的菜鸟，你赢得了一点点喘息的空间和选择的权利。

阶段二：胜利阶段：精心布局，赢得整场战争

当你不再为下个月的开销而焦虑，当你开始拥有稳定的客户和现金流，当你有了几个可以信赖的战友时，恭喜你，你已经成功从生存阶段毕业了。

但危险也随之而来。很多人死在了从生存到胜利的过渡期。因为他们习惯了野猫的生存方式，却不知道如何像狮子一样去统治一片草原。

进入胜利阶段，意味着游戏规则的彻底改变。你不能再用游击战的思维去打阵地战。你需要升级你的整个作战系统。

在这个阶段：

战略开始比战术更重要。你要从低头捡钢镚，转变为抬头看地图。你要思考的不再是“下一单在哪里”，而是“我未来的战场在哪里”。你要开始做减法，砍掉那些虽然能赚钱但没有长期价值的业务。你要开始说“不”，拒绝那些不符合你战略方向的客户和机会。

品牌和声誉成为你的护城河。在生存阶段，你卖的是产品；在胜利阶段，你卖的是信任。你要开始像爱护自己的眼睛一样，爱护你的品牌。你要建立标准，提升品质，提供超出预期的服务。因为品牌是你未来能够轻松赢得战争的重型武器，它能帮你吸引到最优秀的人才和最优质的客户。

系统和团队成为你的放大器。你不能再依靠自己一个人的“野蛮”去冲锋陷阵。你要从一个超级士兵，转变为一个能够排兵布阵的将军。你要找到比你更聪明的人，并赋予他们权力和信任。你要建立流程、制度和文化，让你的军队能够在你不在场的情况下，依然高效运转，自动扩张。

个人成长成为企业的核心引擎。战争的边界，取决于将军的视野。在胜利阶段，你最大的瓶颈就是你自己。你必须不断学习，提升自己的认知。你要学会管理压力，学会处理复杂的人际关系，学会做出那些决定生死的重大选择。你要从一个“做事的人”，蜕变为一个“成事的人”。

从生存到胜利，是从混乱到秩序，从机会主义到长期主义，从单打独斗到组织作战的进化。它要求你亲手杀死一部分过去的自己——那个对所有机会都来者不拒的自己，那个凡事都亲力亲为的自己，那个只相信眼前利益的自己。这个过程是痛苦的，但却是迈向真正强大的必经之路。

这本书将清晰地为你划分这两个阶段，并为你提供在不同阶段下，完全不同的作战地图和武器装备。

三、本书是你的武器库，不是心灵鸡汤

市面上充满了各种“心灵鸡汤”式的商业书籍。它们会告诉你“相信自己，你就能成功”，“追随你的激情，财富自然会来”。这些话语温暖、动人，像一杯冬日里的热可可，能给你带来短暂的慰藉。

但当你面对催缴房租的房东，面对银行发来的催款短信，面对员工期待发工资的眼神时，这些鸡汤除了让你感觉更无助之外，毫无用处。

所以，让我把话说清楚：这本书不是心灵鸡汤。

它是一个武器库。

它不会给你安慰，它只会给你武器。它不会告诉你“一切都会好起来的”，它会告诉你“情况很糟，这是你的刀，这是你的盾，这是敌人的弱点，现在去战斗”。

在第一部分“生存法则篇”，你会找到最直接、最粗暴的求生工具。如何像猎犬一样嗅出商机？如何用“先找买家”原则，在零成本的情况下启动你的第一笔生意？如何处理那些让你头疼的拒绝和谈判？如何管理你那少得可怜的现金？这些都是你可以在战场上立刻使用的格斗技巧和求生装备。它们不优雅，但能救你的命。

在第二部分“赢家心法篇”，你的武器将得到全面升级。你将学会如何打造个人形象这件“无形铠甲”，如何构建强强联合的“人际关系矩阵”，如何运用概率权做出“重大战略选择”，如何管理压力这个“心理军工厂”，以及如何实现自我成长这个“终极武器的锻造”。这些是重型火炮和空天支援，能帮你奠定胜局。

这本书里没有模棱两可的建议，只有清晰的行动指令。它会强迫你直面那些最让你不舒服的问题，会给你一些看似“反常识”但极其有效的策略。阅读它的过程可能不会让你感到愉悦，甚至会让你感到被冒犯。因为它会像一个严厉的教官，不断地撕开你的借口，戳破你的幻想，逼迫你走出舒适区。

不要指望在这里找到一夜暴富的秘诀，因为那不存在。战争的胜利，从来都是靠一场又一场艰苦的战斗堆积起来的。这本书能给你的，不是一张藏宝图，而是一套能让你在任何险恶环境中都能杀出一条血路的作战能力。

所以，在你翻开下一页之前，请问自己一个问题：

你准备好打仗了吗？

如果你只是想寻找一些安慰，或者听一个激动人心的故事，那么请你合上这本书。它不适合你。

但如果你已经准备好，抛弃所有不切实际的幻想，用最务实的态度，最坚韧的神经，去面对这场属于你自己的战争；如果你渴望的不是一句廉价的“加油”，而是一套能让你真正活下来、并最终赢下来的实战方法论——

那么，欢迎来到“野蛮秩序”的世界。

现在，检查你的弹药，握紧你的武器。

让我们开始吧。
